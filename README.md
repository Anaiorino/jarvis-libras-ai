# Jarvis Libras AI 🤟🧠

Sistema de IA capaz de reconhecer sinais em Libras e traduzir para texto e voz em tempo real.

## Tecnologias utilizadas
- Python
- OpenCV
- MediaPipe
- TensorFlow/Keras
- LSTM
- NumPy
- CustomTkinter
- Text-to-Speech

## Funcionalidades
- Reconhecimento de Libras em tempo real
- Tradução para texto
- Voz automática
- Interface moderna dark mode
- Tradução texto → Libras
- Dataset próprio criado manualmente

## Status
🚧 Projeto em andamento

## Objetivo
Criar um sistema multimodal de acessibilidade utilizando Visão Computacional e Inteligência Artificial.
